# 🎯 AI Resume-Based Interview Agent

A smart AI-powered interview system that:

## 🚀 Features
- Resume + JD based question generation
- Adaptive difficulty selection
- Skill-based interview flow
- Zero-score for irrelevant answers
- Final AI evaluation report
- Modular agent architecture (LLM-ready)

---

## 🧠 Architecture

Resume + JD  
→ Skill Extraction  
→ Question Agent  
→ Interview Loop  
→ Evaluator Agent  
→ Feedback Agent
